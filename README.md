# 🌦️ Weather API - .NET 8 Minimal API

Este projeto é um microserviço desenvolvido para consultar dados climáticos em tempo real, focado em **Eficiência Operacional** e **Assincronia**.

## 🚀 O que este projeto demonstra:
- **Integração com APIs Externas:** Consumo de dados reais da Open-Meteo.
- **Performance:** Uso de `HttpClientFactory` para gestão de recursos e `async/await` para evitar travamentos de CPU.
- **Arquitetura Moderna:** Desenvolvido com .NET 8 Minimal APIs, a estrutura mais leve do ecossistema Microsoft.

## 🛠️ Como executar localmente
1. Clone o repositório.
2. No terminal, execute: `dotnet run`.
3. Acesse no navegador: `http://localhost:5168/clima/recife`.

## 🚀 Tecnologias
- **.NET 8** (Minimal APIs)
- **HttpClient Factory** (Gestão eficiente de recursos)

- **Open-Meteo API** para dados climáticos reais.
